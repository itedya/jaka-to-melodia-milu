# Jaka To Melodia - Tampermonkey Script

## Instalacja

+ Zainstaluj wtyczkę [tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
+ W prawym górnym rogu kliknij ikonkę wtyczki, po czym wybierz "Dodaj nowy skrypt"

![Visualisation](https://i.imgur.com/OUPO594.png)

+ Wklej kod pierwszego skryptu 

![Visualisation](https://i.imgur.com/jw4LHjZ.png)

+ Kliknij Plik -> Zapisz

![Visualisation](https://i.imgur.com/h0h6nTa.png)

+ Identycznie dodaj drugi skrypt

+ By wyłączyć skrypt kliknij przycisk w taki sposób jak na obrazku

![Visualisation](https://i.imgur.com/hnzJbRu.png)
